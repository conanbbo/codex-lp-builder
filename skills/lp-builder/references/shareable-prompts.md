# 한국형 LP 제작용 공유 프롬프트

## 히어로 이미지 프롬프트

```text
Create a high-converting Korean mobile landing page hero visual for [상품/서비스].
Target audience: [타깃].
Mood: clean, trustworthy, modern Korean mobile commerce design.
Layout: vertical smartphone landing page slice, strong headline area, product/service visual, clear CTA space.
Avoid tiny unreadable text, distorted hands, fake logos, watermarks.
```

## 문제 공감 섹션

```text
Create a Korean mobile landing page section showing the customer's pain points for [타깃].
Use a clean editorial design, emotional but not exaggerated, with simple icon-like visual elements.
Leave enough blank space for HTML text overlay.
```

## 신뢰 근거 섹션

```text
Create a trust-building mobile landing page section for [상품/서비스].
Include visual placeholders for reviews, numbers, expert profile, or portfolio proof.
Do not invent real brand logos or fake certification marks.
```

## 최종 CTA 섹션

```text
Create a persuasive final CTA section for a Korean mobile landing page.
Mood: confident, warm, action-oriented.
Include visual space for one primary CTA button and short reassurance text.
Avoid legal-risk claims and exaggerated guarantee wording.
```
