# 한국형 LP 질문 은행

## 1. 사업/상품

질문: 어떤 상품이나 서비스를 소개하는 LP인가요?

추천안:
1. 온라인 강의/교육 프로그램
2. 상담·컨설팅 서비스
3. 오프라인 매장/예약 서비스
4. SaaS/앱/디지털 도구
5. 화장품·건강식품·생활용품

## 2. 목적

질문: 이 LP에서 사용자가 최종적으로 어떤 행동을 하길 원하시나요?

추천안:
1. 신청서 작성
2. 카카오톡 문의
3. 전화 상담 예약
4. 결제/구매
5. 자료 다운로드

## 3. 타깃

질문: 가장 먼저 설득하고 싶은 고객 한 명은 누구인가요?

추천안:
1. 바쁜 직장인
2. 1인 사업자/프리랜서
3. 자영업자/소상공인
4. 학부모
5. 취업/이직 준비생

## 4. 오퍼

질문: 고객에게 정확히 무엇을 제공하나요?

추천안:
1. 본강의 + 실습자료 + 템플릿
2. 1:1 상담 + 진단 리포트
3. 체험판 + 정식 구매 안내
4. 예약 혜택 + 할인 쿠폰

## 5. CTA

질문: CTA 버튼 문구는 무엇이 좋을까요?

추천안:
1. 무료 상담 신청하기
2. 지금 강의 신청하기
3. 카카오톡으로 문의하기
4. 자료 무료로 받기
5. 예약 가능 시간 확인하기

## 6. 문제/욕구

질문: 고객이 지금 가장 답답해하는 문제는 무엇인가요?

추천안:
1. 시간이 부족하다
2. 방법을 몰라 시작하지 못한다
3. 비용 대비 효과가 불안하다
4. 기존 방법으로 성과가 나지 않는다
5. 혼자 하려니 오래 걸린다

## 7. 차별점

질문: 비슷한 상품과 비교해 가장 강한 차별점은 무엇인가요?

추천안:
1. 실무형 예제 제공
2. 바로 적용 가능한 템플릿 제공
3. 짧은 시간 안에 결과물 완성
4. 전문가 피드백 포함
5. 초보자도 따라 하기 쉬움

## 8. 신뢰 근거

질문: 고객이 믿을 수 있는 근거는 무엇인가요?

추천안:
1. 후기/리뷰
2. 수강생/고객 수
3. 포트폴리오/사례
4. 자격/경력
5. 언론/기관/파트너 이력

## 9. 디자인 톤

질문: 어떤 분위기의 LP를 원하시나요?

추천안:
1. 신뢰감 있는 기업형
2. 감성적이고 따뜻한 스타일
3. 강한 전환형 세일즈 페이지
4. 미니멀하고 고급스러운 스타일
5. 밝고 친근한 한국형 모바일 상세페이지

## 10. 유입 경로

질문: 방문자는 주로 어디에서 들어오나요?

추천안:
1. 인스타그램 광고
2. 네이버 검색/블로그
3. 유튜브 설명란
4. 카카오톡 공유
5. 문자/이메일 캠페인

## 11. 법적 주의

질문: 특히 조심해야 할 업종이나 표현이 있나요?

추천안:
1. 건강/의료/다이어트
2. 화장품/미용
3. 금융/투자/수익
4. 교육/자격증/취업
5. 특별히 없음
