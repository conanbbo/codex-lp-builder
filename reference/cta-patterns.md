# 한국형 CTA 패턴 모음

CTA는 랜딩페이지의 최종 행동입니다. 이 키트에서는 기본적으로 하나의 CTA만 중심으로 사용합니다.

## 1. 업종별 추천 CTA

| 업종 | 1순위 CTA | 대체 CTA |
|---|---|---|
| 강의/교육 | 무료 상담 신청하기 | 커리큘럼 받아보기, 체험 수업 신청하기 |
| 온라인 클래스 | 지금 수강 신청하기 | 샘플 강의 보기, 강의 안내 받기 |
| 병원/클리닉 | 상담 예약하기 | 가능 여부 확인하기, 전화 상담하기 |
| 미용/피부/다이어트 | 맞춤 상담 신청하기 | 프로그램 안내 받기 |
| B2B/SaaS | 도입 문의하기 | 데모 신청하기, 제안서 받기 |
| 로컬 매장 | 방문 예약하기 | 위치 확인하기, 전화 문의하기 |
| 전자책/자료 | 자료 무료로 받기 | 샘플 다운로드하기 |
| 컨설팅 | 진단 신청하기 | 견적 문의하기, 1:1 상담 신청하기 |
| 세미나/웨비나 | 참가 신청하기 | 일정 알림 받기 |
| 채용/모집 | 지원하기 | 모집 요강 확인하기 |

## 2. CTA 문구 작성 원칙

좋은 CTA:

- 무료 상담 신청하기
- 커리큘럼 받아보기
- 1:1 진단 신청하기
- 카카오톡으로 문의하기
- 데모 일정 잡기
- 자료집 다운로드하기

피해야 할 CTA:

- 확인
- 제출
- 클릭
- 자세히
- 바로가기
- 문의

## 3. 카카오톡 CTA

카카오톡 CTA를 사용할 때 확인할 것:

- 실제 채널 URL 또는 오픈채팅 URL
- 상담 가능 시간
- 자동 응답 여부
- 개인정보를 받는 경우 동의 문구 필요 여부

예시 문구:

```text
카카오톡으로 1분 상담하기
상담 가능 시간: 평일 10:00~18:00
```

## 4. 전화 CTA

전화 CTA를 사용할 때 확인할 것:

- `tel:` 링크 형식
- 통화 가능 시간
- 부재 시 대체 문의 방법

예시:

```html
<a href="tel:010-0000-0000">전화 상담하기</a>
```

## 5. 폼 CTA

폼 CTA를 사용할 때 확인할 것:

- 수집 항목
- 이용 목적
- 보유 기간
- 개인정보 처리방침 URL
- 동의 체크박스

## 6. CTA 통일 규칙

- 히어로 CTA, 중간 CTA, 하단 고정 CTA, 최종 CTA는 같은 행동으로 통일합니다.
- 보조 CTA가 필요하면 시각적 우선순위를 낮춥니다.
- CTA 링크가 미확정이면 `확인 필요`로 표시하고 배포하지 않습니다.
