# 한국형 LP 개인정보·표시광고·후기 문구 템플릿

이 문서는 법률 자문을 대체하지 않습니다. 실제 공개 전에는 사업자, 업종, 수집 정보, 판매 방식에 맞게 전문가 검토가 필요할 수 있습니다.

## 1. 개인정보 수집 동의 기본 문구

폼을 사용하는 경우 아래 정보를 반드시 확인합니다.

```text
개인정보 수집 및 이용에 동의합니다.
수집 항목: 이름, 연락처, 문의 내용
이용 목적: 상담 및 문의 응대
보유 기간: 상담 완료 후 3개월 또는 관련 법령에 따른 보관 기간
```

처리방침 URL이 있는 경우:

```html
<label>
  <input type="checkbox" required>
  개인정보 수집 및 이용에 동의합니다.
</label>
<a href="개인정보처리방침_URL">개인정보 처리방침 보기</a>
```

처리방침 URL이 없는 경우:

```text
확인 필요: 개인정보 처리방침 URL이 필요합니다. 공개 배포 전 반드시 추가하세요.
```

## 2. 과장 표현 금지·주의 목록

| 위험 표현 | 이유 | 대체 표현 |
|---|---|---|
| 100% 보장 | 결과 단정 | 만족도를 높이기 위한 체계적 지원 |
| 무조건 성공 | 개별 결과 차이 무시 | 성공 가능성을 높이는 실습 구조 |
| 국내 최고 | 객관 근거 필요 | 차별화된 강점 |
| 업계 1위 | 객관 자료 필요 | 많은 고객이 선택한 |
| 완치 | 의료적 효능 단정 | 개선에 도움을 줄 수 있는 관리 |
| 부작용 없음 | 안전성 단정 | 개인에 따라 차이가 있을 수 있음 |
| 수익 보장 | 투자·사업 성과 단정 | 실제 사례 기반 성과 예시 |
| 합격 보장 | 교육 결과 단정 | 합격 가능성을 높이는 학습 설계 |

## 3. 후기 문구 기준

안전한 후기 사용 조건:

- 실제 고객 후기인지 확인
- 필요한 경우 익명 처리
- 과도한 성과 일반화 금지
- 개인 경험임을 명시

예시:

```text
수강생 개인의 경험이며, 결과는 개인의 상황과 실행 정도에 따라 달라질 수 있습니다.
```

금지:

```text
모든 수강생이 같은 결과를 얻었습니다.
```

## 4. 성과 수치 문구 기준

수치를 쓰려면 아래가 필요합니다.

- 측정 기간
- 측정 대상
- 계산 기준
- 출처 또는 내부 자료 여부

예시:

```text
2026년 1월~3월 수강 후 설문 응답자 82명 기준, 87%가 “업무 적용에 도움이 되었다”고 답했습니다.
```

확인 필요 문구:

```text
확인 필요: 성과 수치의 출처, 기간, 표본 수가 필요합니다.
```

## 5. 업종별 주의

### 의료/건강

- 치료, 완치, 질병명 개선 표현 주의
- 전후 사진 사용 시 근거와 고지 필요
- 개인차 고지 필요

### 미용/화장품

- 의약품처럼 보이는 효능 표현 주의
- 피부 개선, 재생, 치료 등 표현 검토 필요

### 금융/투자

- 수익 보장, 원금 보장, 손실 없음 표현 금지
- 투자 판단 책임과 위험 고지 필요

### 교육/취업

- 합격 보장, 취업 보장 표현 주의
- 실제 합격률이나 취업률은 기준과 출처 필요

### 전자상거래

- 가격, 배송, 환불, 기간, 조건을 명확히 표시
- 할인 마감이 실제가 아니면 사용 금지

## 6. 배포 전 법적 확인 문구

완료 보고에 아래 항목을 포함합니다.

```text
법적/운영 확인 필요:
- 개인정보 처리방침 URL
- 후기/성과 수치의 실제 근거
- 가격/환불/기간 조건
- 업종별 제한 표현 검토
- 사업자 정보 또는 문의처 표시 필요 여부
```
