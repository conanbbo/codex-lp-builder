# AI로 모바일 LP를 만들 때의 주의사항

## 1. 이미지와 HTML 역할 분리

AI 이미지는 분위기, 배경, 인물, 제품 시각화에 강합니다.
하지만 가격, 날짜, 긴 설명, 법적 고지, CTA 링크는 HTML로 처리하는 것이 안전합니다.

## 2. 한국어 텍스트 주의

GPT Image 2.0은 한국어 표현이 좋아졌지만, 긴 문장이나 작은 글자는 여전히 오타가 생길 수 있습니다.
따라서:

- 이미지 안 문구는 짧게
- 최종 문구는 HTML 텍스트로
- 생성 후 반드시 눈으로 확인

## 3. 가짜 요소 금지

- 실제 존재하지 않는 인증 마크
- 허위 언론 로고
- 근거 없는 후기
- 가짜 수상 이력
- 실존 브랜드 로고 무단 사용

위 요소는 만들지 않습니다.

## 4. 성능

- 이미지 용량을 줄입니다.
- WebP 사용을 고려합니다.
- 불필요한 애니메이션을 줄입니다.
- 모바일 첫 화면 로딩을 우선합니다.

## 5. 접근성

- 이미지에 alt를 넣습니다.
- 핵심 정보를 이미지에만 넣지 않습니다.
- 버튼 텍스트는 명확히 작성합니다.
- 대비가 낮은 색 조합을 피합니다.

## 6. 최종 점검

공개 전에는 실제 스마트폰 화면에서 확인합니다.

- 첫 화면 이해 가능 여부
- CTA 클릭 가능 여부
- 폼 제출 가능 여부
- 링크 오류 여부
- 법적 고지 누락 여부
