# 연결 가이드 — 이미지 생성과 배포

## 이미지 생성

기본은 Codex 내장 GPT Image 2.0을 사용합니다.

권장 방식:
- 먼저 `lp-spec.md`로 섹션 구조를 확정합니다.
- 각 섹션을 모바일 세로형 슬라이스로 나눕니다.
- 긴 한국어 문구는 이미지에 박지 않고 HTML 텍스트로 둡니다.
- 생성 후 오타, 손/얼굴 왜곡, 로고 오류를 확인합니다.

## 배포 선택지

### 1. GitHub Pages

장점:
- 무료
- 정적 HTML LP에 적합
- GitHub 저장소와 연결하기 쉬움

주의:
- 공개 저장소라면 파일이 외부에 보입니다.
- 도메인 연결은 별도 설정이 필요합니다.

### 2. Vercel

장점:
- 배포가 빠름
- 프리뷰 URL 확인이 쉬움
- 정적 사이트와 프론트엔드 프로젝트에 적합

주의:
- 계정 연결이 필요합니다.
- 도메인, 환경변수, 폼 연동은 별도 확인이 필요합니다.

## 배포 전 확인

- 사용자가 배포 방식을 선택했는가?
- 최종 `index.html`이 QA를 통과했는가?
- CTA 링크가 실제 목적지로 연결되는가?
- 개인정보 수집이 있다면 개인정보 처리방침 URL이 있는가?
- 결제, 예약, 신청 폼은 실제 운영 가능한가?
