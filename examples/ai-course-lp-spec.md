# 예시 lp-spec.md — AI 업무 자동화 강의

이 파일은 교육·실습용 예시입니다. 실제 프로젝트에서는 사용자 인터뷰를 통해 새로 확정하세요.

## 1. 프로젝트 개요

- 사업/상품: 직장인을 위한 ChatGPT 업무 자동화 2시간 실습 강의
- LP 목적: 무료 상담 신청 유도
- 핵심 CTA: 무료 상담 신청하기

## 2. 타깃

- 대표 고객 1명: 반복 문서 작업과 보고서 작성에 시간을 많이 쓰는 30대 사무직
- 현재 문제: ChatGPT를 써보고 싶지만 어디에 적용해야 할지 모름
- 원하는 결과: 내 업무에 바로 쓰는 프롬프트와 자동화 흐름을 만들고 싶음
- 구매/신청을 망설이는 이유: 너무 어렵거나 실무에 안 맞을까 봐 걱정

## 3. 오퍼

- 제공 내용: 2시간 실습 강의, 업무별 프롬프트 템플릿, 실습 자료
- 가격/조건: 확인 필요
- 보너스/혜택: 강의 후 복습 자료 제공
- 마감/희소성: 실제 일정 확정 전에는 마감 표현 사용 금지

## 4. 핵심 메시지

- 한 줄 약속: 반복 업무에 바로 적용할 수 있는 AI 실습 흐름을 함께 완성합니다.
- 주요 혜택 3개:
  1. 이메일, 보고서, 회의록 작업에 바로 적용
  2. 초보자도 따라 할 수 있는 단계별 실습
  3. 수업 후 다시 쓸 수 있는 템플릿 제공
- 차별점: 이론보다 실습 중심
- 신뢰 근거: 강사 경력, 수강 후기, 기존 교육 사례는 실제 자료 확인 필요

## 5. 섹션 구성

1. 히어로/FV
2. 업무 비효율 문제 공감
3. 강의 소개
4. 실습 결과물
5. 커리큘럼
6. 수강 대상
7. 후기/사례(실제 근거 필요)
8. FAQ
9. 무료 상담 CTA

## 6. 디자인 방향

- 톤앤매너: 신뢰감 있는 실무형, 밝고 선명한 교육 페이지
- 색감: 흰색 배경, 블루 포인트, 넉넉한 여백
- 이미지 스타일: 노트북으로 실습하는 직장인, 강의 화면, 업무 템플릿 느낌
- 참고 URL: 확인 필요

## 7. 기능 요구사항

- CTA 링크: 확인 필요
- 폼/문의 방식: 상담 신청 폼 또는 카카오톡 문의
- 카카오톡/전화/이메일: 확인 필요
- 추적 태그: 확인 필요

## 8. 법적·운영 주의사항

- 금지 표현: “업무 시간 100% 단축”, “무조건 자동화 성공”, “수익 보장”
- 필수 고지: 결과는 개인 업무 환경과 실행 정도에 따라 달라질 수 있음
- 개인정보 처리: 이름, 연락처, 문의 내용 수집 시 동의 문구 필요
- 확인 필요: 가격, 일정, 신청 링크, 개인정보 처리방침 URL

## 9. 승인 상태

- 사용자 승인: 예시 파일이므로 실제 프로젝트에서 재승인 필요
- 남은 확인 사항: 실제 가격, 일정, CTA 링크, 후기 근거
