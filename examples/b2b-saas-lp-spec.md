# 예시 lp-spec.md — B2B SaaS 데모 신청 LP

이 파일은 교육·실습용 예시입니다. 실제 프로젝트에서는 사용자 인터뷰를 통해 새로 확정하세요.

## 1. 프로젝트 개요

- 사업/상품: 중소기업용 내부 문서 검색 AI 솔루션
- LP 목적: 데모 신청 유도
- 핵심 CTA: 데모 신청하기

## 2. 타깃

- 대표 고객 1명: 사내 문서 검색과 인수인계 문제로 시간을 쓰는 50인 규모 회사의 팀장
- 현재 문제: 자료가 여러 곳에 흩어져 있어 필요한 문서를 찾는 데 오래 걸림
- 원하는 결과: 보안 걱정 없이 내부 자료를 빠르게 검색하고 싶음
- 구매/신청을 망설이는 이유: 보안, 도입 비용, 기존 시스템 연동이 걱정됨

## 3. 오퍼

- 제공 내용: 문서 검색 AI 데모, 도입 상담, 보안/연동 확인
- 가격/조건: 기업 규모와 도입 범위에 따라 별도 문의
- 보너스/혜택: 무료 데모 또는 진단 제공 여부 확인 필요
- 마감/희소성: 사용하지 않음

## 4. 핵심 메시지

- 한 줄 약속: 흩어진 사내 문서를 더 빠르게 찾고 활용할 수 있는 AI 검색 환경을 제안합니다.
- 주요 혜택 3개:
  1. 내부 문서 검색 시간 절감 기대
  2. 팀별 권한과 보안 조건 확인 가능
  3. 도입 전 데모로 실제 사용 흐름 확인
- 차별점: B2B 도입 상담과 보안 검토 중심
- 신뢰 근거: 고객사, 보안 인증, 도입 사례는 실제 자료 확인 필요

## 5. 섹션 구성

1. 히어로/FV
2. 사내 문서 검색 문제 공감
3. 솔루션 소개
4. 사용 시나리오
5. 보안/권한 관리
6. 데모 신청 절차
7. FAQ
8. 최종 데모 신청 CTA

## 6. 디자인 방향

- 톤앤매너: 신뢰감 있는 B2B, 차분하고 전문적인 분위기
- 색감: 네이비, 화이트, 블루 포인트
- 이미지 스타일: 대시보드, 문서 검색 UI, 팀 협업 장면
- 참고 URL: 확인 필요

## 7. 기능 요구사항

- CTA 링크: 데모 신청 폼 URL 확인 필요
- 폼/문의 방식: 회사명, 이름, 이메일, 연락처, 문의 내용
- 카카오톡/전화/이메일: 이메일 문의 가능 여부 확인 필요
- 추적 태그: 확인 필요

## 8. 법적·운영 주의사항

- 금지 표현: “검색 시간 100% 절감”, “완벽한 보안”, “무조건 비용 절감”
- 필수 고지: 도입 효과는 문서 환경, 데이터 구조, 연동 범위에 따라 달라질 수 있음
- 개인정보 처리: 데모 신청 폼의 개인정보 동의 및 처리방침 URL 필요
- 확인 필요: 보안 인증, 고객사 로고 사용 허가, 도입 사례 근거

## 9. 승인 상태

- 사용자 승인: 예시 파일이므로 실제 프로젝트에서 재승인 필요
- 남은 확인 사항: 실제 기능 범위, 보안 자료, 데모 링크, 고객사 사용 허가
