# 예시 lp-spec.md — 로컬 프로필 촬영 예약 LP

이 파일은 교육·실습용 예시입니다. 실제 프로젝트에서는 사용자 인터뷰를 통해 새로 확정하세요.

## 1. 프로젝트 개요

- 사업/상품: 이직·브랜딩용 프로필 촬영 패키지
- LP 목적: 카카오톡 예약 문의 유도
- 핵심 CTA: 카카오톡으로 예약 문의하기

## 2. 타깃

- 대표 고객 1명: 이직 준비 중이라 깔끔한 프로필 사진이 필요한 30대 직장인
- 현재 문제: 급하게 사진이 필요하지만 어디서 찍어야 자연스럽고 믿을 만한지 모름
- 원하는 결과: 부담 없이 상담하고 일정과 가격을 빠르게 확인하고 싶음
- 구매/신청을 망설이는 이유: 보정이 과하거나 가격이 불명확할까 봐 걱정

## 3. 오퍼

- 제공 내용: 프로필 촬영, 기본 보정, 원본 제공 여부 확인 필요
- 가격/조건: 확인 필요
- 보너스/혜택: 예약자 혜택은 실제 여부 확인 필요
- 마감/희소성: 실제 예약 가능 시간 기준으로만 사용

## 4. 핵심 메시지

- 한 줄 약속: 자연스럽고 신뢰감 있는 프로필 사진을 편하게 준비하세요.
- 주요 혜택 3개:
  1. 용도에 맞춘 촬영 방향 상담
  2. 자연스러운 표정과 포즈 안내
  3. 예약 전 가격과 구성 확인 가능
- 차별점: 직장인 프로필 목적에 맞춘 상담형 촬영
- 신뢰 근거: 포트폴리오, 실제 리뷰, 스튜디오 위치 확인 필요

## 5. 섹션 구성

1. 히어로/FV
2. 프로필 사진 고민 공감
3. 촬영 패키지 소개
4. 촬영 진행 과정
5. 포트폴리오/예시
6. 가격/구성
7. 위치/운영 시간
8. FAQ
9. 카카오톡 예약 CTA

## 6. 디자인 방향

- 톤앤매너: 깔끔하고 따뜻한 스튜디오 느낌
- 색감: 아이보리, 차콜, 부드러운 포인트 컬러
- 이미지 스타일: 자연광 스튜디오, 인물 촬영 무드
- 참고 URL: 확인 필요

## 7. 기능 요구사항

- CTA 링크: 카카오톡 채널 URL 확인 필요
- 폼/문의 방식: 카카오톡 문의 우선
- 카카오톡/전화/이메일: 확인 필요
- 추적 태그: 확인 필요

## 8. 법적·운영 주의사항

- 금지 표현: “무조건 인생사진”, “100% 만족”, “최저가 보장”
- 필수 고지: 촬영 결과는 개인 컨디션, 촬영 조건, 보정 범위에 따라 달라질 수 있음
- 개인정보 처리: 카카오톡 문의에서 이름/연락처/예약 정보를 받을 경우 안내 필요
- 확인 필요: 실제 가격, 환불/변경 규정, 위치, 예약 가능 시간

## 9. 승인 상태

- 사용자 승인: 예시 파일이므로 실제 프로젝트에서 재승인 필요
- 남은 확인 사항: 실제 포트폴리오, 가격, 카카오톡 링크, 예약 정책
