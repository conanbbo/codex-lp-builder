Korean Mobile LP Builder for Codex

한국 실무자가 바로 판매·교육·배포에 쓸 수 있는 모바일 LP 제품판

사전 인터뷰로 사양을 고정하고, 한국형 CTA·개인정보·표시광고·모바일 QA 기준을 적용해 Codex에서 랜딩페이지 제작 흐름을 끝까지 운영합니다.

사용 흐름 보기
1. 사전 인터뷰

한 번에 하나씩 질문해 타깃, 오퍼, CTA, 법적 제약을 확정합니다.

2. lp-spec.md

확정된 사양을 계약서처럼 사용해 제작 범위를 고정합니다.

3. 제품판 QA

기능, 디자인, 모바일, 카피, 법적 리스크, 배포 준비까지 점검합니다.

100점 제품판에 추가된 것

  • 업종별 CTA 패턴: 강의, 병원, B2B, 로컬 매장, 전자책, 컨설팅
  • 개인정보 수집 동의, 후기·성과 수치, 과장 광고 대체 문구 템플릿
  • 390px/430px 기준 모바일 디자인 시스템
  • GitHub Pages/Vercel 배포 전 체크리스트
  • 교육·검수용 테스트 시나리오와 예시 lp-spec.md 3종

Codex에서 시작하는 문장

한국형 모바일 LP 만들어줘.
먼저 lp-grilling으로 사전 인터뷰를 진행하고,
질문은 한 번에 하나씩만 해줘.
내 답변을 바탕으로 lp-spec.md를 만든 뒤,
내가 승인하면 lp-builder로 제작과 QA를 진행해줘.

중요: 파일 하나가 아니라 폴더 전체를 Codex에서 열어야 합니다.
